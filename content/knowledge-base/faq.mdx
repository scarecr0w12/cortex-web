# Frequently Asked Questions

## What is CortexPrism?

CortexPrism is an open-source agentic harness system that provides a runtime for building, deploying, and managing AI agents. It supports multiple LLM providers, features a five-tier memory system, and includes a comprehensive security model.

## Do I need a GPU?

No, CortexPrism runs entirely on CPU. LLM calls are made through provider APIs, so you only need a network connection.

## What providers are supported?

OpenAI, Anthropic, Google AI, Groq, Azure OpenAI, AWS Bedrock, Together AI, Fireworks, Mistral AI, Perplexity, DeepSeek, and more.

## Can I use it offline?

CortexPrism requires network access for LLM API calls. However, the memory system and plugin management work offline.
