# Multi-Agent Orchestration

Coordinating multiple agents to solve complex tasks through structured collaboration strategies. Introduced in v0.51.0.

## 6 Orchestration Strategies

| Strategy | Description | Best For |
|----------|-------------|----------|
| **Sequential** | Agents execute in order, each receiving prior output | Linear workflows |
| **Parallel** | Multiple agents execute simultaneously on subtasks | Decomposable tasks |
| **Debate** | Agents argue opposing positions, judge synthesizes | Decision-making |
| **Review-Loop** | Executor produces, critic reviews, revises until approved | Quality-critical outputs |
| **Hierarchical** | Manager decomposes, delegates to workers, synthesizes | Complex multi-step projects |
| **Graph (DAG)** | Tasks arranged as dependency graph with parallel branches | Mixed dependencies |

## 13 Sub-Agent Types

Explorer, Coder, Researcher, Planner, Generalist, Security Auditor, Debugger, Architect, DevOps, Data Analyst, UI/UX Designer, Code Reviewer, Technical Writer
