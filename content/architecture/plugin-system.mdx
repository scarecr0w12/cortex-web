# Plugin System

CortexPrism supports three types of plugins: ESM modules, MCP servers, and WebAssembly modules.

## Plugin Types

| Type | Description | Use Case |
|------|-------------|----------|
| ESM | JavaScript/TypeScript modules | General-purpose plugins |
| MCP | Model Context Protocol servers | Tool providers |
| WASM | WebAssembly modules | Performance-critical tasks |

## Plugin Lifecycle

1. **Install**: Download and register the plugin
2. **Enable**: Activate the plugin in the current session
3. **Use**: Call plugin capabilities through the tool system
4. **Disable**: Deactivate without uninstalling
5. **Uninstall**: Remove the plugin entirely
