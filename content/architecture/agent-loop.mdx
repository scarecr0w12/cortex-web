# Agent Loop

The agent loop is the core execution engine of CortexPrism.

## Execution Flow

1. Receive user input
2. Retrieve relevant context from memory
3. Build prompt with system instructions and context
4. Send to LLM via model router
5. Parse response for tool calls
6. Execute approved tool calls
7. Store results in working memory
8. Return response to user

## Configuration

The agent loop is configurable via the agent configuration system:

- `maxTurns`: Maximum conversation turns
- `temperature`: Response randomness
- `tools`: Available tools for the agent
- `systemPrompt`: Custom system instructions
