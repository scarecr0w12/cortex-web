# `cortex start` (removed in v0.52)

The `cortex start` command was removed in v0.52.0. Server and daemon management are now separate:

| Old Command | New Command |
|-------------|-------------|
| `cortex start` | [`cortex daemon start`](/docs/cli/daemon) + [`cortex server start`](/docs/cli/serve) |
| `cortex start --server-only` | [`cortex server start`](/docs/cli/serve) |
| `cortex start --daemon-only` | [`cortex daemon start`](/docs/cli/daemon) |

See the [CLI Reference](/docs/cli) for the full command list.
