# `cortex restart` (removed in v0.52)

The `cortex restart` command was removed in v0.52.0. Server and daemon management are now separate:

| Old Command | New Command |
|-------------|-------------|
| `cortex restart` | [`cortex daemon restart`](/docs/cli/daemon) + `cortex server restart` |
| `cortex restart --server-only` | `cortex server restart` |
| `cortex restart --daemon-only` | [`cortex daemon restart`](/docs/cli/daemon) |

See the [CLI Reference](/docs/cli) for the full command list.
