# `cortex stop` (removed in v0.52)

The `cortex stop` command was removed in v0.52.0. Server and daemon management are now separate:

| Old Command | New Command |
|-------------|-------------|
| `cortex stop` | [`cortex daemon stop`](/docs/cli/daemon) + [`cortex server stop`](/docs/cli/serve) |
| `cortex stop --server-only` | `cortex server stop` |
| `cortex stop --daemon-only` | [`cortex daemon stop`](/docs/cli/daemon) |

See the [CLI Reference](/docs/cli) for the full command list.
