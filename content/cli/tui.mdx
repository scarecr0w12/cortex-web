# `cortex agent tui`

Launch the Terminal User Interface (TUI) for Cortex, providing an interactive streaming chat experience in the terminal.

## Usage

```bash
cortex agent tui
```

## Overview

The TUI provides a split-pane terminal interface for interacting with Cortex:
- **Left pane**: Streaming chat with the agent
- **Right pane**: Tool calls and command history
- Keyboard-driven navigation and command input

## Examples

```bash
# Launch the terminal UI
cortex agent tui
```
