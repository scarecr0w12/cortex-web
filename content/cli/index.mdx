# CLI Reference

CortexPrism provides a comprehensive command-line interface for interacting with the agentic system.

## Usage

```bash
cortex <command> [options]
```

## Global Options

| Option | Description |
|--------|-------------|
| `--config, -c` | Path to config file |
| `--data-dir, -d` | Data directory path |
| `--verbose, -v` | Enable verbose logging |
| `--help, -h` | Show help |
| `--version, -V` | Show version |

## Available Commands

- `cortex chat` — Start an interactive chat session
- `cortex serve` — Start the HTTP API server
- `cortex daemon` — Manage the background daemon
- `cortex run` — Execute scripts and commands
- `cortex memory` — Memory system management
- `cortex reflect` — Agent reflection and self-analysis
- `cortex vault` — Encrypted secrets management
- `cortex policy` — Security policy management
- `cortex jobs` — Scheduled job management
- `cortex sessions` — Session management
- `cortex setup` — Initial setup wizard
