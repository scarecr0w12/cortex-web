# CLI Reference

CortexPrism provides a comprehensive command-line interface for interacting with the AI Agent Operating System. All commands are invoked through the `cortex` binary.

## Usage

```bash
cortex <command> [options]
```

## Global Options

| Option | Environment Variable | Description |
|--------|---------------------|-------------|
| `--config, -c` | `CORTEX_CONFIG` | Path to config file (default: `~/.cortex/config.json`) |
| `--data-dir` | `CORTEX_DATA_DIR` | Data directory path (default: `~/.cortex/`) |
| `--verbose, -v` | â€” | Enable verbose logging |
| `--help, -h` | â€” | Show help |
| `--version, -V` | â€” | Show version |

## Command Groups

### Agent

| Command | Description |
|---------|-------------|
| [`cortex agent chat`](/docs/cli/chat) | Interactive streaming chat session with LLM providers |
| [`cortex agent exec`](/docs/cli/agent-exec) | Execute agent tasks non-interactively (one-shot) |
| [`cortex agent tui`](/docs/cli/tui) | Terminal UI |
| [`cortex agent sessions`](/docs/cli/sessions) | List and manage chat sessions |
| [`cortex agent eval`](/docs/cli/eval) | Evaluation framework for benchmarking agent performance |
| [`cortex agent reflect`](/docs/cli/reflect) | Inspect and consolidate reflection patterns |
| [`cortex agent lint`](/docs/cli/agentlint) | Audit agent configs, tools, plugins, and prompts |
| [`cortex agent voice`](/docs/cli/voice) | Voice mode management (enable, disable, set-voice, set-speed) |

### Server

| Command | Description |
|---------|-------------|
| [`cortex server start`](/docs/cli/serve) | Start the HTTP + WebSocket server with web UI (was `cortex serve`) |

| `cortex server restart` | Restart the HTTP server |
